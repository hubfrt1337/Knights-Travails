Knight Travails
How to Use?
Open console => (right click then select inspect, in the panel that opens click the 'Console' tab at the top)
To create graph, use: const graph = new Graph(8)
To visualize how the chessboard looks, use: `console.log(graph.matrix);`
To find the shortest path for the knight, use the command: graph.knightMoves([start], [end]);
Both start and end take two arguments: the x and y coordinates, ranging from [0, 0] to [7, 7]. The function returns an array representing the shortest possible path the knight can take.
